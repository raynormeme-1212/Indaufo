
IndaUFO Starter Site
====================

This is a small static starter site for IndaUFO (Dx2c). 
Includes index.html, styles.css, script.js and sample images in /assets.

How to deploy to Netlify (fast):
1. Zip this folder OR push it to a GitHub repo.
2. Create a free Netlify account (https://app.netlify.com).
3. Click 'Add new site' -> 'Upload site' and drag the contents of this folder (index.html + assets + css/js).
4. Netlify will provide a free temporary URL like https://example-name.netlify.app
5. To connect your own domain (unusualsoundz.com) - follow Netlify's 'Domain settings' and add your custom domain.

Calendly + BeatStars + Stripe:
- Replace placeholder embed areas with your actual embed codes from Calendly and BeatStars.
- For payments, set up Stripe and add Checkout links to Buy buttons or integrate with server-side code.

Contact info to update in index.html: replace [your-email@example.com] with actual email.

Starter files included:
- index.html
- styles.css
- script.js
- assets/studio_hero.jpg (from uploaded images)
- assets/studio_gallery.jpg (from uploaded images)
